# Anu

Toolkit for data-driven dynamic Babylonjs scene graphs. 

## Quick Start
```
git clone <repo url>
cd anu
npm install 
npm run build
```

## Linking to another node project
```
npm link <path to project you want to link to>
```

## Read the WIP docs
```
cd docs
npm install 
npm run docs:dev
```

### Everytime after you can run this command from the root folder instead

```
npm run docs
```
